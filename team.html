Code submit

Teams that want to be considered for the prizes must submit the code they wrote to generate their solutions, for the jury to judge. We kindly ask you to host the code elsewhere and send us a link, e.g. a Github repository, Google Drive folder, Dropbox link, etc. We'll try to access this in the days after the contest, so it is fine to send us the repository link during the contest but only make it public after the contest has ended (and we will contact you in case we can't download the code).

To make jury life easier, please include a small README giving an overview of the solution and which parts were written during the first 24 hours of the contest.
